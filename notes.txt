Notes

- In Sublime, can save html file, type html and press Tab to generate the following boilerplate code:
<html>
<head>
	<title></title>
</head>
<body>

</body>
</html>

- In sublime, to toggle comment on a line, enter Cmd + /
- The <p> tag will block text together and add spacing after
- Instead of <b>, use <strong>. Instead of <i>, use, <em> (for emphasis)
- In sublime, to duplicate a line enter Cmd + Shift + D
- In sublime, can type tag name then tab to autocomplete, e.g strong + Tab = <strong></strong>
